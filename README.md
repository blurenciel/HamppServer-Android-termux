# INSTALLATION

```
pkg update -y && pkg upgrade -y
termux-setup-storage
pkg install git -y
git clone https://github.com/yourchocomate/HamppServer.git
cd HamppServer
chmod +x install
./install
or
bash install
```

# USAGE
Type `hampp` and start using HamppServer :)
